客户主数据系统
================

## 前言
    此次项目为某大型房地产集团的外包项目《客户主数据系统》由于数据量庞大，导致系统性能较差，所以集团提出重构意向，并要求对系统使用基于vue.js的前后端分离；所以这次用了Vue + Element组件库做后台管理系统
    现学现卖完成的项目，所以存在很多不足，但是也让我学到了很多，所以在此记录本次项目的代码，留作日后学习之用

## 功能
* Element UI
* 登录/退出/修改密码
* 表格
* 表单
* 点击编辑后在新标签页打开
* 复选按钮
* 全选/全反选
