实地客户主数据系统
================

## 项目介绍
    此次项目为实地集团的外包项目《客户主数据系统》
    由于数据量庞大，导致系统性能较差，所以实地集团提出重构意向，并要求对系统使用基于vue.js的前后端分离<br>
    所以这次用了Vue + Element组件库做后台管理系统
