本地启动方式
进入项目根目录

npm install

npm run dev

目录结构
* build - webpack config files
* config - webpack config files
* dist - build
* src 
    * api
    * assets
    * common
    * components
    * mock
    * styles
    * views
    * vuex
    * App.vue
    * main.js
    * routes.js
* static - static assets
